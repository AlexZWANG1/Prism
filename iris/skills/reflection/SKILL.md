# Reflection & Learning

Use when the user asks to review predictions, validate past analysis, or discover patterns.

## Single-company review
1. Recall prior analysis and predictions
2. Fetch latest actual data
3. Compare each key metric: predicted vs actual, calculate deviation
4. For each significant deviation: what method was used, why it was wrong, what to do next time
5. Save experiences with methodology details
6. Update company memory notes

## Trade review
1. Read closed trades from portfolio
2. Analyze: win rate, by confidence tier, by sector
3. Check process discipline: did thesis-broken positions exit promptly? Were stops honored?
4. Save statistical experiences

## Pattern discovery
1. Recall experiences for a sector
2. Look for repeated patterns across companies
3. Summarize into reusable methodology templates
4. Save as pattern-level experiences
