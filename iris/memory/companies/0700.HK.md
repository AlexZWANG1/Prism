# Tencent (0700.HK) 初步分析
- 用户请求：分析腾讯控股。
- 受限条件：当前运行环境缺少实时财务抓取工具与用户上传财报资料，因此估值基于近似分部与保守假设，属于方向性框架，不应视为精确目标价。
- DCF 假设：游戏/增值服务 3200亿，广告 1200亿，金融科技与企业服务 2300亿；WACC 10%，永续增长 3%，毛利率 49%，Opex/Revenue 24%，Capex/Revenue 8%，税率 18%。
- DCF 输出：每股公允价值约 198.95（相对现价 380，隐含约 -47.6%）。敏感性区间约 144-335。
- Comps：数据质量较差，腾讯与部分港股/ADR同业前瞻估值字段缺失，无法形成强有力相对估值结论；仅见腾讯毛利率高于部分可比中位数。
- 初步结论：若现价约 380 港元，则在该保守框架下腾讯更接近“价格不便宜/可能高估”，但由于输入未用最新财报校准，置信度中等偏低。
- 后续优化重点：用最新年报/季报校准真实收入分部、资本开支、净现金、股本，并重跑 DCF。
